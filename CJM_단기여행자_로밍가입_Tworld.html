Action
K 여행 일정을 확정하고 목적지 국가와 체류 기간(3~7일)을 결정함. 주변 지인이나 SNS에서 "해외에서 데이터 어떻게 씀?"을 검색·질문함. 최근 1년 이내 해외 방문 경험 있는 SKT 사용자 2,700명 (전체 응답자 6,000명의 45%) ※ CSI 2025 하반기 Table L-1 / 임원 인터뷰 9 (로밍 상품 포트폴리오)
K 클럽 T로밍 인지도: SKT 사용자 중 31.1%만 인지, 전체(타사 포함) 21.3%. 5G 사용자는 상대적으로 인지도 높음(20~22%). LTE 사용자 인지도는 10~11%에 그침 ※ CSI 2025 상반기 Table I-18-1
S 로밍 vs 현지 유심 vs 포켓 와이파이 중 어떤 게 유리한지 네이버·유튜브에서 비교 정보를 찾아봄. 출국 전 SKT 로밍 가입이 필요하다는 것을 인지함
출처: SKT CSI 2025 / 임원 인터뷰 9 (로밍 상품 포트폴리오) / Web 검색 리서치
K Tworld 앱 또는 웹사이트를 열어 로밍 메뉴를 찾으려 함. 메인 화면에서 '로밍·해외' 카테고리를 탐색하며 진입 경로를 파악함. 20~30대 고객의 매장 방문 감소 추세에 따라 앱 셀프서비스 의존도 증가 (임원 인터뷰 11)
S 메뉴 구조가 복잡해 '로밍' 메뉴를 즉시 찾지 못하고 검색창에 "로밍"을 직접 타이핑함. Tworld 앱의 하단 탭이나 메인 배너를 통해 우회 진입 시도
출처: SKT 임원 인터뷰 11 (20~30대 비대면 증가 트렌드) / Tworld UI 리서치
K 데이터 로밍 상품 목록(baro 요금제, 데이터 로밍 쿠폰, 음성 로밍 등)을 확인하고 일별 가격과 데이터 한도를 비교함. 여행 일수에 따른 총 비용을 직접 계산해봄
S 단기(1~7일) 플랜과 월정액 플랜의 가격 차이를 인지. 상품 설명에서 '망내 할인', '망외 요금' 등 통신 전문 용어를 이해하기 어려워 반복해서 읽음
출처: 로밍 가격 비교 리서치 (단기 플랜 30% 비쌈) / 임원 인터뷰 7 (서비스 복잡성)
K 원하는 로밍 상품을 선택 후 '가입하기' 버튼을 탭함. 본인 인증 과정(PASS 앱 또는 SMS 인증)을 거쳐 최종 신청을 완료함
S 가입 신청 중 SIM 보안 서비스 가입 여부 확인 팝업이 등장하거나, 특정 부가서비스 해지 필요 안내가 뜨는 경우 당황하여 프로세스가 중단됨
출처: Web 리서치 (SIM 보안서비스 충돌 이슈) / 임원 인터뷰 2 (앱 사용 불편 위험)
K Tworld에서 가입 완료 SMS 또는 앱 알림을 확인함. 단말기 설정에서 '데이터 로밍 허용'을 켜고, 자동 업데이트·백업 앱의 와이파이 전용 설정으로 전환함
S 설정 방법을 Tworld에서 안내받지 못해 유튜브나 네이버 블로그에서 별도 검색("아이폰 로밍 설정 방법", "갤럭시 로밍 설정"). 출국 당일 공항에서 부랴부랴 확인하는 경우 많음
출처: Web 리서치 (출국 전 설정 불안) / SKT 뉴스룸 로밍 가이드
K 도착 후 자동으로 현지 통신망에 연결되길 기대하며 스마트폰을 켬. 데이터가 연결되면 카카오톡·구글 맵을 사용하고, 연결이 안 되면 문제를 직접 해결하려 시도함
S 일부 국가에서 자동 망 접속 실패 시 수동으로 네트워크를 검색하거나 비행기 모드를 껐다 켬. 데이터 사용량이 얼마나 됐는지 실시간으로 알 수 없어 불안함을 느낌
출처: Web 리서치 (수동 네트워크 전환, 실시간 요금 확인 불가) / SKT Roamight 가이드
K 귀국 후 Tworld 앱에서 로밍 요금 청구 내역을 확인함. 예상보다 높은 요금이 청구된 경우 고객센터에 문의하거나 청구서 상세 내역을 살펴봄
S 로밍 상품이 자동으로 만료되지 않거나, 해지 방법을 몰라 다음 달도 과금될까 봐 Tworld에서 수동 해지 처리를 진행함
출처: Web 리서치 (자동해지 미처리로 재청구) / SKT CSI 데이터
💬 Feeling
감정
K "로밍 요금이 얼마나 나올지 모르겠어, 그냥 현지 유심 사는 게 나을까?"
S 정보가 너무 많고 옵션도 많아 선택 피로를 느낌. 잘못 선택하면 큰 요금이 나올 것 같다는 막연한 불안감
K "로밍 메뉴가 어디 있지? 통신 앱은 왜 이렇게 복잡하게 만들어 놨지?"
S Tworld 앱이 기능은 많은데 원하는 걸 찾기 어렵다는 답답함. 첫 화면부터 로밍 배너가 보이면 좋겠다는 생각
K "baro 요금제가 뭔데? 일일 요금이랑 묶음 요금이랑 어떤 차이야?"
S "3박 4일인데 4일짜리 끊어야 해? 3일짜리 끊어야 해? 출국일도 포함되는 건가?"
K "신청하려니까 뭔가 다른 서비스 해지하라고 하는데… 그냥 해지하면 괜찮은 건가?"
S 신청 도중 예상치 못한 팝업이나 조건이 등장하면서 신뢰감 저하. "이거 해도 되나?" 하는 불확실감
K "가입은 됐는데 설정을 제대로 한 건지 모르겠어. 현지 가서 데이터가 안 되면 어쩌지?"
S 출발 당일 공항에서도 설정 확인을 반복하며 불안. Tworld가 단계별 설정 가이드를 제공하지 않아 스스로 해결해야 한다는 부담감
K "데이터 얼마나 썼지? 요금 한도 초과하면 어떻게 되는 거야?"
S 연결은 됐지만 속도가 느리거나 끊기면 "내가 설정을 잘못한 건지, 원래 이런 건지" 구분이 안 돼 답답함. 여행에 집중하지 못하고 핸드폰 화면만 들여다보게 됨
K "청구서에 로밍 요금이 기대보다 많이 나왔네. 어떤 항목에서 이렇게 나온 거지?"
S 해지를 안 하면 다음 달에도 요금이 붙을까봐 걱정. 자동 해지가 됐는지 확인하기 위해 Tworld를 재접속함
⚡ Pain
point
K SKT 로밍 vs 현지 유심 vs 포켓 와이파이 등 대안 정보가 SKT 공식 채널 밖에 더 풍부해, SKT 채널 이탈이 시작되는 시점
S SKT 앱·웹에서 '여행 전 로밍 준비' 콘텐츠 노출이 부족. 개인화된 추천 없이 일반 배너만 노출됨
K 온·오프라인 채널 간 일관성 부족. 로밍 메뉴 구조가 직관적이지 않아 탐색 단계에서 이탈률 높음
출처: SKT 임원 인터뷰 11 (T멤버십-Tworld 연결성 낮음)
S Tworld 앱이 한국어 전용 구조로, 로밍처럼 해외 관련 정보가 필요한 상황에서 영어 전환 등 글로벌 UX 부재
K 상품이 많고 이름이 유사(baro 데이터 1일, baro 데이터 3일 등)해 선택 혼란 유발. 내 요금제에 따른 추천 기능 없음
출처: SKT 임원 인터뷰 7 (서비스 복잡도 높음)
S 단기 플랜은 월정액 대비 약 30% 비싸지만 비교 기준 없이 가격만 나열됨. '며칠 여행이면 어떤 게 유리한지' 계산기 미제공
K SIM 보안 서비스 등 특정 부가서비스 가입 상태에서 로밍 신청 시 충돌 발생. 사용자가 어떤 서비스를 해지해야 하는지 명확히 안내받지 못함
출처: Web 리서치 (SIM 보안-로밍 충돌 이슈)
S 가입 완료 후 확인 SMS가 즉시 오지 않거나 앱에서 가입 상태가 바로 반영되지 않아 이중 신청 우려
K 가입 완료 후 단말 설정 방법이 Tworld 앱 내에서 제공되지 않아 사용자가 외부에서 정보를 찾아야 함
출처: SKT 임원 인터뷰 6 (디지털 채널 사후 만족도 하락)
S 아이폰·갤럭시 별 설정 방법이 다름에도 기종별 맞춤 가이드 미제공. '데이터 로밍 켜기'만 안내하고 앱 자동업데이트 차단 등 실비용 절감 팁 없음
K 실시간 데이터 사용량 및 요금 확인이 불가능 (해외 통신사의 과금 데이터 지연 전송 문제). 한도 초과 알림이 늦게 오거나 안 옴
출처: SKT 임원 인터뷰 7 (레거시 시스템 한계) / Web 리서치
S 현지 망 연결 실패 시 자동 복구 기능 없음. 비행기 모드 껐다 켜기, USIM 재삽입 등 사용자가 직접 트러블슈팅 해야 하는 불편
K 로밍 요금 항목(데이터/음성/문자 각각)이 청구서에서 명확히 구분되지 않아 과다 청구 여부 판단 어려움. 해외 로밍 할인 서비스 만족도 평균 7.91/100 (최저 4.2 ~ 최고 10.2, 세그먼트별 편차 큼)
출처: SKT CSI 2025 상반기 (해외 로밍 할인 항목) / 임원 인터뷰 6
S 로밍 상품이 자동 만료되는 상품과 수동 해지가 필요한 상품이 혼재하여 혼란. 해지 후에도 요금이 청구되는 사례 발생

K 가입 전 사전 점검 기능
"내 부가서비스 중 로밍 가입에 영향을 주는 게 있는지" 자동으로 체크하고 원클릭 정리
S [잠재 Needs] 가입 신청 후 즉각적인 상태 확인 (앱 내 실시간 가입 상태 표시). SMS가 아닌 앱 푸시로 즉시 알림
K 기종별 단말 설정 가이드 제공
가입 완료 직후, 아이폰/갤럭시 기종을 인식해 맞춤 설정 체크리스트를 앱 내에서 바로 제공
S [잠재 Needs] "현지에서 불필요한 데이터 소모 방지" 팁(앱 자동업데이트 OFF, iCloud 백업 와이파이 전용 등)까지 포함한 절약 가이드
K 실시간 데이터 사용량 및 요금 현황 표시
Tworld 앱에서 "현재까지 ○○원 사용 중" 형태로 실시간 모니터링 기능 제공
S [잠재 Needs] 연결 실패 시 앱 내 원클릭 자가해결(트러블슈팅) 가이드. "지금 데이터가 안 되시나요?" 버튼 하나로 진단·해결
K 로밍 요금 항목별 투명한 청구서
데이터·음성·문자별 사용량과 요금을 시각화한 청구서. "어디서 얼마 썼는지" 날짜별 상세 내역
S [잠재 Needs] 귀국 감지 후 로밍 상품 자동 해지 알림 또는 원클릭 해지. "귀국하셨나요? 로밍 정리하기" 푸시

I 여행 정보 연동 로밍 큐레이션
항공 예매 앱(네이버·카카오 등)과 연동하거나, 사용자 캘린더에서 출국 일정을 감지해 Tworld가 선제적으로 "○월 ○일 출국 예정, 로밍 준비하세요" 알림을 발송하는 AI 에이전트형 알림 서비스
I 로밍 전용 숏컷 & 스마트 진입 UX
Tworld 앱 홈에 '✈ 로밍 준비' 고정 아이콘 배치. 출국 D-7 이내 접속 시 로밍 준비 팝업 자동 노출. 검색창 "로밍" 타이핑 시 즉시 상품 리스트로 딥링크
I 여행 일정 기반 로밍 상품 시뮬레이터
목적지 국가·여행 기간·사용 패턴(SNS 중심 vs 지도 내비 중심 vs 영상 스트리밍 등)을 입력하면 "예상 요금 ○○원, 추천 상품 ○○" 형태로 최적 플랜을 추천. 경쟁사 현지유심 비용과도 투명 비교 제공
I 가입 前 충돌 서비스 자동 진단 & 해결 플로우
"가입하기" 클릭 시 먼저 내 부가서비스를 스캔해 충돌 요소를 알려주고, "○○ 서비스를 임시 해지 후 로밍 가입 → 귀국 후 자동 재가입" 원클릭 플로우 제공
I 가입 완료 후 '출국 준비 체크리스트' 인앱 가이드
로밍 가입 완료 직후 "3단계 출국 준비" 카드형 가이드 표시. ① 단말 데이터 로밍 ON ② 앱 자동업데이트 OFF ③ 이메일·클라우드 와이파이 전용 설정. 기종(iOS/Android) 자동 감지 후 스크린샷 기반 안내
I 실시간 로밍 미터 & AI 절약 코치
Tworld 앱 위젯으로 "오늘 ○○MB 사용 / 잔여 ○○원" 실시간 표시. 속도 저하·연결 오류 감지 시 자동 트러블슈팅 가이드 푸시. "현재 위치에서 무료 Wi-Fi 근처" 안내 기능 연동
I 귀국 감지 자동 해지 + 투명 청구서
국내 통신망 재연결 감지 시 "귀국하셨나요? 로밍 서비스 정리하기" 푸시 1회 발송 후 원클릭 해지. 청구서는 여행지별·날짜별·항목별 시각화 리포트로 제공. 다음 여행을 위한 "이번 여행 로밍 총 비용 ○○원, 더 저렴한 옵션이 있어요" 피드백 제공
